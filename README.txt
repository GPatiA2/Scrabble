-- Para crear un proyecto en Eclipse que use la carpeta SRC que hay en el repositorio

	--1. Clonad el repositorio en vuestro ordenador
	--2. Cread un nuevo proyecto de Eclipse que esté en la ubicación de la carpeta donde habeis clonado el repositorio
	--Así, se añade automáticamente en el .gitignore la carpeta /bin/ y teneis en la carpeta del proyecto de Eclipse la
		carpeta SRC del repositorio + el resto de carpetas que no tienen ficheros de código.
	--Revisad el .gitignore por si Git no ha incluido automáticamente en él los ficheros .project y .classpath del proyecto

--Los archivos necesarios para ejecutar el programa son las carpetas del src, lib y Dibujos

-- Para ejecutar el programa hay dos caminos:
	-- VESION NORMAL: Run>>main y por ahi elegir consola o Gui
	-- VERSION MULTIJUGADOR LOCAL: 1. run>>servidor
				       2. run>>cliente
